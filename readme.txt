# Scientific Calculator

This project is a simple web-based scientific calculator built using HTML, CSS, and JavaScript. It provides basic arithmetic operations like addition, subtraction, multiplication, and division.

## Features

- Basic arithmetic operations: add, subtract, multiply, divide.
- Clear input for new calculations.
- User-friendly interface.

## Getting Started

These instructions will get you a copy of the project up and running on your local machine for development and testing purposes.

### Prerequisites

What things you need to install the software:

- A modern web browser like Chrome, Firefox, Safari, or Edge.

### Installing

A step by step series of examples that tell you how to get a development environment running:

1. Clone the repository to your local machine or download the ZIP file and extract it.
2. Navigate to the project directory.
3. Open the `index.html` file in your web browser.

## Usage

To use the calculator:

1. Click on the number buttons on the interface to enter numbers.
2. Click on the operation buttons (`+`, `-`, `*`, `/`) to choose an operation.
3. Click the `=` button to get the result.
4. Use the `C` button to clear the input field and perform new calculations.

## Built With

- HTML - For structuring the web page.
- CSS - For styling the calculator interface.
- JavaScript - For the calculator's functionality.
